Bite Decoded is a responsive web project designed to "deconstruct" the nutritional facts behind popular fast food and comfort meals. Rather than boring spreadsheets, this project visualizes macros and ingredients using a playful "fast food" aesthetic combined with clean data typography.

The goal is to make nutritional transparency accessible, engaging, and easy to digest.

Responsive Layout: Fully optimized for desktop and mobile viewing.

## Mapbox setup

This project can use Mapbox GL for the interactive map in `src/components/Map.js`.

- Sign up for a free Mapbox account at https://www.mapbox.com/ and get an access token.
- Create a `.env` file in the project root with this line (never commit secrets to git):

```
REACT_APP_MAPBOX_TOKEN=your_mapbox_token_here
```

After adding the token, restart the dev server (`npm start`) so Create React App picks up the environment variable.

If you prefer not to use Mapbox, remove the token and the map component will skip initialization.
